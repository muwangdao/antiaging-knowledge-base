## 🇨🇳 中国衰老标志物研究联合体（ABC）共识专区

### 关于 ABC

**中国衰老标志物研究联合体（Aging Biomarker Consortium, ABC）** 是由中国科学院动物研究所刘光慧研究员、裴钢院士等领衔发起的全国性衰老生物标志物研究协作组织，旨在建立适合中国人群的衰老评估体系和干预策略。

**核心成果**：
- 📊 脂肪组织衰老生物标志物框架（2025 年 7 月）
- 💪 骨骼肌衰老生物标志物共识（2025 年 1 月）
- 🐒 灵长类衰老生物标志物系统框架（2025 年 9 月，Nature Reviews Molecular Cell Biology）
- 🎯 X-Age 项目：中国人群综合衰老评估体系

### 1️⃣ 脂肪组织衰老生物标志物框架

<div id="abc-adipose" style="background: linear-gradient(135deg, #fff5f5 0%, #fed7d7 100%); padding: 30px; border-radius: 12px; margin: 30px 0; border: 2px solid #f56565;">

#### ABC 脂肪组织衰老共识（2025 年 7 月）

**参考文献**：
- Yu J, Zhang Y, Zhang T, et al. A framework of biomarkers for adipose tissue aging: a consensus statement by the Aging Biomarker Consortium. *Life Med*. 2025;4(5):lnaf027. PMID: [41089668](https://pubmed.ncbi.nlm.nih.gov/41089668/)
- DOI: [10.1093/lifemedi/lnaf027](https://doi.org/10.1093/lifemedi/lnaf027)

**共识要点**：

脂肪组织是人体重要的能量储存和代谢器官。随着中国老年人口激增，脂肪组织衰老成为多器官衰老的关键起始事件。ABC 专家组通过文献综述和专家意见，建立了脂肪组织衰老生物标志物的系统框架。

#### 三大维度评估框架

|--------|------|---------|-----------|---------|

| **脂联素** | 体液 | ⭐⭐⭐ 高 | ⭐⭐⭐ 高 | ⭐⭐⭐ 强烈推荐 |
| **脂肪细胞大小** | 结构 | ⭐⭐ 中 | ⭐⭐ 中 | ⭐⭐ 推荐 |
| **纤维化标志物** | 结构 | ⭐⭐ 中 | ⭐⭐ 中 | ⭐⭐ 推荐 |
| **IL-6/TNF-α** | 体液 | ⭐⭐ 中 | ⭐⭐⭐ 高 | ⭐⭐ 推荐 |

**中英文摘要**：

> **中文**：脂肪组织作为人体重要的能量储存和代谢器官，其衰老是多器官衰老的关键起始事件。本共识从功能、结构和体液三个维度系统推荐了脂肪组织衰老评估标志物，为临床和科研提供了标准化评估框架。
>
> **English**: As a crucial energy storage and metabolic organ, adipose tissue aging represents a pivotal initiating event for multi-organ senescence. This consensus systematically recommends biomarkers for adipose tissue aging assessment across functional, structural, and humoral dimensions, providing a standardized evaluation framework for clinical and research applications.

[→ 查看完整共识原文](https://doi.org/10.1093/lifemedi/lnaf027)

### 2️⃣ 骨骼肌衰老生物标志物共识

<div id="abc-muscle" style="background: linear-gradient(135deg, #f0f7ff 0%, #e1effe 100%); padding: 30px; border-radius: 12px; margin: 30px 0; border: 2px solid #4299e1;">

#### ABC 骨骼肌衰老共识（2025 年 1 月）

**参考文献**：
- Aging Biomarker Consortium. A framework of biomarkers for skeletal muscle aging: a consensus statement by the Aging Biomarker Consortium. *Life Med*. 2025;3(6):lnaf001. PMID: [40008206](https://pubmed.ncbi.nlm.nih.gov/40008206/)
- DOI: [10.1093/lifemedi/lnaf001](https://doi.org/10.1093/lifemedi/lnaf001)

**共识要点**：

骨骼肌衰老导致肌少症（sarcopenia）和衰弱（frailty），严重影响老年人生活质量。ABC 专家组从功能、影像及体液三个维度推荐了骨骼肌衰老评估指标。

**中英文摘要**：

> **中文**：骨骼肌衰老导致的肌少症和衰弱严重影响老年人生活质量。本共识从功能、影像及体液三个维度系统推荐了骨骼肌衰老评估指标，为肌少症的早期诊断和干预提供了标准化评估工具。
>
> **English**: Skeletal muscle aging leads to sarcopenia and frailty, significantly impacting the quality of life in the elderly. This consensus systematically recommends assessment indicators for skeletal muscle aging across functional, imaging, and humoral dimensions, providing standardized evaluation tools for early diagnosis and intervention of sarcopenia.

### 3️⃣ 灵长类衰老生物标志物系统框架

<div id="abc-primate" style="background: linear-gradient(135deg, #faf5ff 0%, #e9d8fd 100%); padding: 30px; border-radius: 12px; margin: 30px 0; border: 2px solid #9f7aea;">

#### ABC 灵长类衰老共识（2025 年 9 月）

**参考文献**：
- Aging Biomarker Consortium. A systematic framework of primate aging biomarkers. *Nat Rev Mol Cell Biol*. 2025 Sep. PMID: 待更新
- DOI: 待更新

**共识要点**：

ABC 团队在《Nature Reviews Molecular Cell Biology》发表灵长类衰老生物标志物系统框架，首次从细胞、组织和个体三个层面系统梳理了人类与非人灵长类的衰老生物标志物，提出了新概念体系。

#### 新概念体系

<div style="display: grid; grid-template-columns: 1fr 1fr 1fr; gap: 20px; margin: 20px 0;">

<div style="background: white; padding: 20px; border-radius: 8px; border-left: 4px solid #9f7aea;">
<h4 style="margin-top: 0; color: #2d3748;">🔬 衰老标志物 (Senomarker)</h4>
<p style="color: #718096; font-size: 14px;">反映衰老状态和进程的生物标志物</p>
<ul style="margin: 10px 0; padding-left: 20px; color: #718096; font-size: 13px;">
<li>表观遗传时钟</li>
<li>端粒长度</li>
<li>衰老相关分泌表型（SASP）</li>
<li>线粒体功能指标</li>
</ul>
</div>

<div style="background: white; padding: 20px; border-radius: 8px; border-left: 4px solid #f56565;">
<h4 style="margin-top: 0; color: #2d3748;">⚡ 衰老激活因子 (Senoactivator)</h4>
<p style="color: #718096; font-size: 14px;">促进或加速衰老进程的因素</p>
<ul style="margin: 10px 0; padding-left: 20px; color: #718096; font-size: 13px;">
<li>慢性炎症</li>
<li>氧化应激</li>
<li>代谢紊乱</li>
<li>干细胞耗竭</li>
</ul>
</div>

<div style="background: white; padding: 20px; border-radius: 8px; border-left: 4px solid #48bb78;">
<h4 style="margin-top: 0; color: #2d3748;">🛡️ 衰老抑制因子 (Senorepressor)</h4>
<p style="color: #718096; font-size: 14px;">延缓或抑制衰老进程的因素</p>
<ul style="margin: 10px 0; padding-left: 20px; color: #718096; font-size: 13px;">
<li>自噬激活</li>
<li>热量限制</li>
<li>运动干预</li>
<li>Senolytics</li>
</ul>
</div>

</div>

#### 三层评估体系

| 层面 | 标志物类型 | 示例 | 应用场景 |
|------|-----------|------|---------|
| **细胞层面** | 分子标志物 | DNA 甲基化、端粒长度、SASP 因子 | 基础研究、药物筛选 |
| **组织层面** | 器官功能 | 肌肉功能、脂肪分布、血管弹性 | 临床评估、干预监测 |
| **个体层面** | 综合指标 | 表型年龄、衰弱指数、认知功能 | 人群研究、健康评估 |

**中英文摘要**：

> **中文**：本框架首次从细胞、组织和个体三个层面系统梳理了人类与非人灵长类的衰老生物标志物，提出了"衰老标志物"、"衰老激活因子"、"衰老抑制因子"的新概念体系，为衰老研究和干预提供了统一的理论框架和评估标准。
>
> **English**: This framework systematically organizes aging biomarkers across cellular, tissue, and individual levels for both humans and non-human primates, proposing a novel conceptual system of "senomarker," "senoactivator," and "senorepressor," providing a unified theoretical framework and evaluation standards for aging research and interventions.

[→ 查看完整共识原文](https://www.nature.com/nrmcb/)

### 4️⃣ X-Age 项目：中国人群综合衰老评估体系

<div id="x-age-project" style="background: linear-gradient(135deg, #f0fff4 0%, #c6f6d5 100%); padding: 30px; border-radius: 12px; margin: 30px 0; border: 2px solid #48bb78;">

#### X-Age 项目

**项目目标**：开发面向中国人群的综合衰老评估体系

**研究内容**：
- 📊 建立中国人群特异性衰老时钟
- 🧬 整合多组学数据（基因组、表观组、蛋白组、代谢组）
- 🏥 开发临床可用的衰老评估工具
- 📈 追踪衰老轨迹和干预效果

**项目进展**：

| 阶段 | 时间 | 目标 | 状态 |
|------|------|------|------|
| **一期** | 2024-2025 | 队列建立与基线数据采集 | ✅ 已完成 |
| **二期** | 2025-2026 | 多组学数据整合分析 | 🔄 进行中 |
| **三期** | 2026-2027 | 衰老时钟开发与验证 | ⏳ 计划中 |
| **四期** | 2027-2028 | 临床工具转化与应用 | ⏳ 计划中 |

**预期成果**：
- ✅ 中国人群表观遗传时钟
- ✅ 多组学整合衰老评分
- ✅ 临床可用衰老评估工具包
- ✅ 衰老干预效果评估标准

**参与机构**：
- 中国科学院动物研究所
- 北京医院
- 上海交通大学医学院附属瑞金医院
- 中山大学附属第一医院
- [更多机构...]

[→ 了解 X-Age 项目详情](https://www.ageingx.cn/)

function searchTrials() {
    const keyword = document.getElementById('trial-keyword').value.toLowerCase();
    const phase = document.getElementById('trial-phase').value;
    const status = document.getElementById('trial-status').value;
    
    let filtered = trialData.filter(t => {
        // 关键词筛选
        if (keyword && !t.intervention.toLowerCase().includes(keyword) && 
            !t.condition.toLowerCase().includes(keyword) &&
            !t.nctId.toLowerCase().includes(keyword)) {
            return false;
        }
        
        // 阶段筛选
        if (phase !== 'all' && t.phase !== phase) return false;
        
        // 状态筛选
        if (status !== 'all' && t.status !== status) return false;
        
        return true;
    });
    
    displayTrials(filtered);
}
