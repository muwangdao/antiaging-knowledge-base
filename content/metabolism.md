## 📊 核心代谢通路

### 1. NAD+ 代谢通路

**通路概述**：
NAD+（烟酰胺腺嘌呤二核苷酸）是细胞能量代谢和信号传导的关键辅因子，随年龄增长而下降。NAD+ 是 Sirtuins 去乙酰化酶和 PARPs DNA 修复酶的必需辅因子。

**关键节点**：
- **NMN**（烟酰胺单核苷酸）：NAD+ 直接前体
- **NR**（烟酰胺核糖）：NAD+ 前体
- **NAMPT**：NAD+ 合成限速酶
- **Sirtuins**：NAD+ 依赖性去乙酰化酶
- **PARPs**：NAD+ 依赖性 DNA 修复酶
- **CD38**：NAD+ 降解酶（随年龄增加）

**通路图**：
```
色氨酸 → Kynurenine → QA → NAD+
     ↓
烟酸 → NAMN → NAAD → NAD+
     ↓
烟酰胺 → NMN → NAD+ → NR（循环）
                ↓
          Sirtuins / PARPs / CD38
```

### 2. mTOR 信号通路

**通路概述**：
mTOR（雷帕霉素靶蛋白）是细胞生长和代谢的核心调控因子，整合营养、能量和生长因子信号。mTOR 过度激活加速衰老，抑制可延长寿命。

**关键节点**：
- **mTORC1**：雷帕霉素敏感，调控蛋白质合成和自噬
- **mTORC2**：雷帕霉素不敏感，调控细胞骨架和存活
- **AMPK**：能量感应器，抑制 mTOR
- **Insulin/IGF-1**：上游激活信号
- **S6K**：mTORC1 下游效应器
- **4E-BP1**：翻译起始调控

**通路图**：
```
Insulin/IGF-1 → PI3K → AKT ─┬→ mTORC1 → S6K / 4E-BP1 → 蛋白质合成
                            └→ mTORC2 → AKT（正反馈）
AMPK（能量低）→ ─| mTORC1（抑制）→ 自噬激活
```

### 3. 胰岛素/IGF-1 信号通路

**通路概述**：
胰岛素和 IGF-1 信号是进化上保守的衰老调控通路。抑制该通路可延长多种生物寿命，但可能影响葡萄糖稳态。

**关键节点**：
- **Insulin Receptor**：胰岛素受体
- **IRS1/2**：胰岛素受体底物
- **PI3K/AKT**：下游信号传导
- **FOXO**：转录因子，调控应激抵抗

### 4. AMPK 能量感应通路

**通路概述**：
AMPK（AMP 活化蛋白激酶）是细胞能量状态的主要感应器，低能量时激活，促进分解代谢，抑制合成代谢。AMPK 激活模拟热量限制效应。

**关键节点**：
- **AMP/ATP 比率**：激活信号
- **LKB1**：上游激酶
- **CaMKKβ**：钙依赖性激活
- **mTOR**：下游抑制靶点
- **PGC-1α**：线粒体生物合成
- **Sirtuins**：协同作用

**通路图**：
```
AMP↑/ATP↓ → AMPK 激活 ─┬→ ─| mTOR（抑制合成）
                       ├→ PGC-1α（线粒体生成）
                       ├→ Sirtuins 激活
                       └→ 自噬激活
```

### 5. 线粒体代谢网络

**通路概述**：
线粒体是能量生产和代谢调控中心，线粒体功能障碍是衰老的核心标志之一。

**关键节点**：
- **TCA 循环**：能量代谢核心
- **电子传递链**：ATP 生产
- **ROS 产生**：氧化应激来源
- **线粒体自噬**：质量控制
- **线粒体动力学**：融合/分裂平衡

**通路图**：
```
葡萄糖 → 丙酮酸 → TCA 循环 → NADH/FADH2
                    ↓
            电子传递链 → ATP + ROS
                    ↓
            线粒体自噬（清除损伤）
```

### 6. 氨基酸代谢与 mTORC1

**通路概述**：
氨基酸（特别是亮氨酸、精氨酸）是 mTORC1 的关键激活信号，连接营养状态与细胞生长。

**关键节点**：
- **亮氨酸**：mTORC1 强激活剂
- **精氨酸**：mTORC1 激活
- **Sestrin2**：亮氨酸感应器
- **CASTOR1**：精氨酸感应器
- **Rag GTPases**：mTORC1 招募
