### 3️⃣ 性能优化

**优化措施**：

| 优化项 | 方法 | 预期提升 |
|-------|------|---------|
| **资源压缩** | Hugo 内置 minify | 体积 -40% |
| **图片优化** | WebP 格式 + 懒加载 | 加载速度 +60% |
| **CSS 内联** | Critical CSS 内联 | FCP -30% |
| **DNS 预取** | 预连接外部资源 | 延迟 -20% |
| **缓存策略** | 浏览器缓存 + CDN | 重复访问 +80% |
| **代码分割** | 按需加载（待实现） | TTI -25% |

**新增配置**：
```toml

  quality = 85
  resampleFilter = CatmullRom

  minifyOutput = true
```

### 5️⃣ 可维护性优化

**配置分离**：

```
config/_default/
├── hugo.toml        # 主配置
├── params.toml      # 站点参数
├── markup.toml      # Markdown 渲染
├── outputs.toml     # 输出格式
├── permalinks.toml  # 永久链接
├── sitemap.toml     # Sitemap 配置
└── robots.toml      # Robots 配置
```

**优势**：
- ✅ 配置清晰，易于查找
- ✅ 模块化管理，降低冲突
- ✅ 支持多环境配置（dev/staging/prod）
